User Management
Create User
用途：
创建User
操作步骤：

打开Users菜单，可通过以下两种方式进入菜单：
Modeler菜单路径	Modeler→Admin→Access Control→Users
快速访问	Users

点击Create按钮，在右侧表单中显示Users新增表单
在新增表单中填写或者选择相应的数据，其中带“★”号的字段为必填项
点击Save按钮，User创建成功
选中一条User数据，点击数据行后的Edit按钮，可在右侧表单中进行Users信息编辑
选中一条User数据，点击数据行后的More-Delete按钮或者Batch Operation-Delete按钮，可删除数据
属性说明：
属性	说明
User Account	用户登录账号
User Name	用户名
Work No	工号
Post	职位，数据来源Position UI
Role Assignment	选择角色，数据来源Roles UI
Department 	Allocation	部门，数据来源SDWT UI
Expiration Policies	过期的政策，数据来源Time Strategy UI
Birthday	生日
Gender	性别
Email	邮件地址
Phone Number	手机号码
Tel	固定电话

结果：
创建/修改/删除User成功
Assign Roles to User
用途：
给User分配Role
操作步骤：
有两种方式，可对User分配Roles，一是在Users页面进行操作，一是在Roles页面进行操作。

打开Users页面，在新增或者编辑User时，在Role Assignment下拉框中，选择Role，可多选
信息填写完整后，点击Save按钮，则给User分配Roles操作成功

打开Roles菜单，可通过以下两种方式进入菜单：
Modeler菜单路径	Modeler→Admin→Access Control→Roles
快速访问	Roles

选中一条Role数据，点击操作列的User按钮，在弹出的页面上，点击Existing Users，在User弹出框中，进行选择
点击OK按钮，给User分配Role成功
属性说明：
属性	说明
Role Code	角色代码
Role Name	角色名称

结果：
User分配Role成功
