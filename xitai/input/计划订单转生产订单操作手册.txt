单个计划订单转生产订单
路径： 后勤 -> 生产 -> 能力计划 -> 需求 -> 生产订单 -> 控制 -> 库存/需求清单 （事务代码：MD04 ）

操作描述

进入初始界面：初始屏幕	1、在首界面输入事务代码：MD04（单个计划订单转生产订单） 进入需求清单界面，在界面中输入需要转换的物料号和工厂代码1200，然后继续按enter进入下一界面
物料号输入	进入界面后，会显示根据MRP跑出的计划订单，双击后会弹出小窗口，点击窗口上的“眼镜”可查看该订单，双击窗口上的“笔”可以更改生产版本，计划订单入库的存储位置，在此界面还可以更改订单的数量，最后返回，确认后点击保存，再次刷新后可以看到计划订单带*号，说明计划订单已被锁定	输入字段说明	输入正确的物料号	 						3、右图为点击“眼镜”进入后查看的界面
4、点击“笔”进入修改界面，可以修改订单数量、生产版本、生产订单入库的存储位置、是否锁定生产订单等	点击保存下方会提示计划订单已修改			输入字段说明	保存：确认修改完成后进行保存。	删除：单击删除，可以删除这个计划订单。	组件ATP：进行可用性检查，齐套后可以进行下单。	总订单数量：可以修改计划订单数量。	结束、开始：计划订单开始日期和结束日期。	存储位置；可以修改建议入库的库存地点。	生产版本：可以修改当前使用的生产版本。	计划订单：勾选后就可以锁定当前计划订单，再运行MRP后不会删除这个计划订单。
5、再点击刷新，会看到更改锁定后的订单会带*号，表示该订单已被锁定，下次不管怎么跑MRP,计划订单永远都会被保留在此界面。
6、双击计划订单再次弹出窗口，点击生产订单，进入生产订单抬头界面。
7、常规视图界面可修改数量，也可对组件和工序进行修改	确认无误后点击保存			输入字段说明	产品信息：工序与组件需核对正确

批量计划订单转生产订单
菜单路径
路径：SAP菜单 -> 后勤 -> 生产 -> 车间现场控制 -> 订单 -> 创建 -> 计划订单的集中转换 (事务代码：CO41)

操作描述
进入已计划订单的集中转换初始界面	在SAP菜单栏内，点击事务代码CO41，按回车，进入显示已计划订单的集中转换初始界面。在此界面根据自己的需求选择，确认后点击运行选择执行	。	输入字段说明	计划工厂：当前需要转的计划订单工厂。	MRP控制员：可以选定指定的产品分类。	物料：可以输入指定物料。	订单类型：当前订单可以批量转的订单类型。	根据状态选择：
可以选择手工审核过的计划订单。	根据可用性选择：可以选择可用性检查完的计划订单，才转生产订单。
2、进入此界面会看到许多需要转生产订单的物料和对应的数量	选中自己需要转换的物料号和数量，点击上方转换，最后可以看到转换后的生产订单号。	输入字段说明	转换：将选定的计划订单转为生产订单。	全选、取消不选：单击后会全选计划订单OR取消全选计划订单。	删除行：将选定的计划行进行删除。	选定行：单击后会将该行显示为橙色。	开始-结束：计划订单开始日期和结束日期，可以进行调整。	订单数量：本次计划订单转换数量，可以进行调整。	订单类型：本次计划订单转生产订单对应的订单类型， 可以进行调整。	工厂：选定的工厂。	订单：转换后的生产订单。
