创建工作中心
菜单路径
路径：SAP菜单 -> 后勤 -> 生产 -> 主数据 -> 工作中心 -> 工作中心 -> 创建 （事务代码：CR01 ）

操作描述

进入工作中心创建的初始界面：初始屏幕	在SAP菜单栏内，输入事务代码CR01，按回车键进入创建工作中心：初始屏幕。
工作中心信息输入	在“工厂”“工作中心”“工作中心类别”输入对应信息，输入完成后按回车键。	工作中心编码规则：
长度8位，工厂编码（4位）+车间（1位）+流水码（3位）	输入字段说明	工厂：1200	工作中心：识别工作中心的代码	工作中心类别：默认0001
创建工作中心：基本数据	输入对应的工作中心名称	输入“负责人”，“用途”，“标准值码”点击回车，再点击默认值页面		输入字段说明	负责人：工作中心负责人	用途：默认001用于控制可使用操作资源的任务清单类型	标准值码：默认ZP01
创建工作中心：缺省值	输入“控制键值”，“标准值的计量单位”,输入字段说明	控制键值：确定应该为属于工作清单或订单（例如排产或成本核算）的对象执行哪些业务事务	标准文本码：通过输入工序中的标准文本码，为工序分配标准文本。
创建工作中心：成本中心分配	输入“成本中心”、“活动类型”、“公式码”，点击回车，再点保存	输入字段说明	成本中心：工作中心所在的车间

修改工作中心
菜单路径
路径：SAP菜单 -> 后勤 -> 生产 -> 主数据 -> 工作中心 -> 工作中心 -> 更改 （事务代码：CR02 ）

操作描述

进入工作中心更改界面	在SAP菜单栏内，点击事务代码CR02，按回车进入更改工作中心界面，在“工作中心”输入对应信息。	输入字段说明	工作中心：识别工作中心的代码
修改工作中心：基本数据	输入对应的工作中心名称	输入“负责人”，“用途”，“标准值码”点击回车，再点击默认值页面		输入字段说明	负责人：工作中心负责人	用途：默认001用于控制可使用操作资源的任务清单类型	标准值码：默认ZP01
修改工作中心：缺省值	输入“控制键值”、“标准值的计量单位”点击回车，再点成本核算页面	输入字段说明	控制键值：确定应该为属于工作清单或订单（例如排产或成本核算）的对象执行哪些业务事务	标准文本码：通过输入工序中的标准文本码，为工序分配标准文本。
修改工作中心：成本中心分配	输入“成本中心”，“活动类型”，“公式码”，点击回车，再点保存	输入字段说明	成本中心：工作中心所在的车间
